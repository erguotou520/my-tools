# My GUI tools

## 特性

- 全局快捷键打开和关闭
- 输入数字可以快速计算
- 输入系统应用名称可以打开应用
- 输入网址则用默认浏览器打开
- 支持文件搜索
- 支持应用插件市场
- 本地配置

## 应用市场

- JSON/yaml 美化
- 有道/谷歌/DeepL翻译，OpenAI翻译
- SSH 隧道代理
- 静态文件服务器，支持路径选择，端口配置，CORS等
- Markdown 文件管理

## 应用开发

- 应用名称，图标，说明 Markdown
- 是否开启
- 可设置全局快捷键打开
- 生命周期：启动前/启动后/退出前/退出后
- 其它配置项
- 前后端代码
